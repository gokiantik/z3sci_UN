🧪 5 SKENARIO UJI + LAGU + FORM FEEDBACK

Simulasi Sarinem (HTML / Opal) — isi feedback untuk bantu pengembangan

👩‍🦱 1. Istri → Suami Dewi, 32 th

🎵 The Winner Takes It All
📋 Dewi dipukul, uang dikuasai, dilarang ketemu keluarga.
#Pesan UserTujuan
1"Halo Sarinem, saya Dewi. Saya takut. Suami saya sering marah-marah dan memukul saya. Saya tidak tahu harus bagaimana."S3/S5, prioritas safety
2"Dia juga mengontrol semua uang. Saya tidak punya tabungan sendiri. Saya ingin keluar tapi tidak punya biaya."Kekerasan ekonomi → S3
3"Saya pernah lapor ke polisi, tapi mereka bilang 'urusan rumah tangga'. Saya semakin takut."Validasi + alternatif
4"Saya punya satu teman yang mau menampung saya. Tapi saya takut anak-anak saya."S8 rencana aman
5"Saya sudah siap. Saya mau keluar hari ini. Tolong bantu saya."Info hotline/shelter

👨 2. Suami → Istri Andi, 40 th

🎵 Usik - Feby Putri
📋 Andi dilempar barang, diancam tuduhan palsu, takut kehilangan anak.
#Pesan UserTujuan
1"Saya Andi, laki-laki. Istri saya sering membentak saya dan melempar piring ke arah saya. Saya malu cerita ke siapa pun."Inklusivitas gender
2"Dia selalu mengancam akan bilang ke polisi bahwa saya yang memukul dia. Saya takut ditangkap."Edukasi KDRT laki-laki
3"Apakah laki-laki bisa jadi korban KDRT? Saya pikir hanya perempuan."Edukasi inklusivitas
4"Saya sudah tidak tahan. Tapi saya takut kehilangan anak-anak jika cerai."Rujuk layanan untuk ayah
5"Saya ingin konseling. Adakah layanan untuk laki-laki korban KDRT?"Rujuk psikolog/LBH

🧒 3. Anak laki → Ayah Rizal, 15 th

🎵 Anakku - Pyurborn
📋 Rizal dipukul ikat pinggang, dimaki, takut karena tidak punya HP sendiri.
#Pesan UserTujuan
1"Halo Sarinem... Saya Rizal, masih 15 tahun. Ayah saya sering memukul saya pakai ikat pinggang. Saya takut."Prioritas keselamatan anak (S5)
2"Saya takut lapor ke polisi. Ayah saya bilang polisi tidak akan percaya anak kecil."Koreksi informasi
3"Apakah saya bisa lapor ke guru? Saya takut ayah saya marah lagi."Dorong bicara ke guru/DP3A
4"Ayah saya baca chat saya. Saya tidak punya HP sendiri, pakai HP ayah."Edukasi safe browsing
5"Saya ingin pergi ke rumah paman. Tapi takut ayah saya cari."Rencana aman + hotline anak 119

👧 4. Anak pr → Ibu Sari, 13 th

🎵 Break The Silence - Audrey V.
📋 Sari dipukul, dikunci, disebut anak gagal, merasa bersalah.
#Pesan UserTujuan
1"Saya Sari, 13 tahun. Ibu saya marah terus dan memukul saya. Hari ini saya tidak boleh keluar kamar."Isolasi + kekerasan fisik → S5
2"Ibu saya bilang saya anak yang gagal, tidak berguna. Saya jadi sedih sekali."Kekerasan verbal → S4 support
3"Apakah ini salah saya? Mungkin saya memang anak yang nakal."Rasa bersalah → reframe
4"Saya ingin telepon hotline anak. Tapi saya takut Ibu dengar."Tips aman menelepon
5"Besok ada guru BK di sekolah. Apakah saya harus cerita ke beliau?"Perkuat keberanian + hak anak

🦋 5. Kupu-Kupu Malam Mawar, PSK

🎵 Kupu-Kupu Malam - Titiek Puspa
📋 Mawar (pekerja seks) dipukuli, ditipu, HP dirusak, stigma polisi.
#Pesan UserTujuan
1"Saya Mawar, kerja di malam hari. Saya dipukuli sama pacar saya, katanya karena saya telat pulang. HP saya juga dia rusak."Kekerasan fisik + HP rusak → S5
2"Saya sudah 3 tahun sama dia. Katanya akan nikahi saya. Saya yang kerja, uangnya dia ambil semua."Eksploitasi ekonomi → S3
3"Saya malu lapor polisi. Saya takut sama polisi, karena pekerjaan saya bukan pekerjaan baik."Stigma ganda → validasi hak
4"Dia pernah menikam paha saya pakai pisau. Bekasnya masih ada. Saya takut dia akan bunuh saya."Senjata tajam → S5, info shelter
5"Saya tidak punya siapa-siapa. Keluarga malu. Teman takut. Saya mau keluar tapi tidak tahu caranya."Isolasi total → S4+S7+S8
